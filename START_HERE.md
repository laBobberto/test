# 🎉 LifeBalance SPb - Проект завершен!

## 📦 Что создано

Полнофункциональный MVP персонального городского ассистента для Санкт-Петербурга.


## ⚡ Быстрый старт

```powershell
# Запустите этот скрипт
.\quick-start.ps1
```

Или вручную:

```bash
# Terminal 1 - Backend
cd backend
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
python init_db.py
uvicorn main:app --reload

# Terminal 2 - Frontend
cd frontend
npm install
npm run dev
```

Откройте: http://localhost:5173

## 🎯 Основные возможности

1. **🤖 AI-планирование** - Генерация персонализированного плана дня через Omniroute
2. **🎮 Геймификация** - Баллы, достижения, стрики за выполнение активностей
3. **🗺️ Карта событий** - Интерактивная карта Leaflet с событиями города
4. **⚖️ Приоритеты** - 6 категорий жизненного баланса (образование, карьера, здоровье, досуг, социализация, быт)
5. **👤 Роли** - Студент, житель, турист с персонализацией
6. **💬 AI чат** - Общение с персональным ассистентом
7. **📊 Статистика** - Отслеживание прогресса и баланса
8. **📱 Адаптивный дизайн** - Работает на всех устройствах

## 🛠️ Технологии

**Backend:**
- FastAPI + Python 3.11
- SQLAlchemy + PostgreSQL/SQLite
- JWT Authentication
- Omniroute AI Integration

**Frontend:**
- React 18 + TypeScript
- Vite + Tailwind CSS
- Zustand + TanStack Query
- Leaflet Maps
- Playwright E2E Tests
