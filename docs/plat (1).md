  Технологический стек                                                             
                                
  Frontend:                     
  - React 18 + TypeScript                                                                                                                                                                     
  - Vite (сборка)                                                                                                                                                                             
  - React Router (навигация)                                                                                                                                                                  
  - Zustand (state management)                                                                                                                                                                
  - TanStack Query (data fetching)                                                                                                                                                            
  - Leaflet (карты)                                                                                                                                                                           
  - Recharts (графики)                                                                                                                                                                        
  - Tailwind CSS (стили)                                                                                                                                                                      
                                                                                                                                                                                              
  Backend:                                                                                                                                                                                    
  - Python 3.11+ / Node.js 20+ (выбор на этапе реализации)                         
  - FastAPI / Express.js                                  
  - PostgreSQL (основная БД) 
  - Redis (кеширование)                                                                                                                                                                       
  - SQLAlchemy / Prisma (ORM)                                                                                                                                                                 
                                                                                                                                                                                              
  AI & Интеграции:                                                                                                                                                                            
  - OpenAI API / Anthropic Claude                                                                                                                                                             
  - Локальные модели (опционально)                                                                                                                                                            
  - REST API клиенты для внешних сервисов                                                                                                                                                     
                                                                                                                                                                                              
  DevOps:                                                                                                                                                                                     
  - Docker + Docker Compose                                                        
  - Git для версионирования                                                                                                                                                                   
  - pytest / Jest для тестирования

  - Настроить REST API framework                                                                                                                                                              
  - Создать базовые модели данных (User, Priority, Activity, Event)

  Database Schema (основные таблицы)                                                                                                                                                          
                                                                                                                                                                                              
  -- Пользователи                                                                                                                                                                             
  users (                                                                                                                                                                                     
      id UUID PRIMARY KEY,                                                                                                                                                                    
      email VARCHAR UNIQUE,                                                                                                                                                                   
      password_hash VARCHAR,                                                                                                                                                                  
      name VARCHAR,                                                                                                                                                                           
      roles JSONB,  -- ['student', 'resident']                                                                                                                                                
      created_at TIMESTAMP                                                                                                                                                                    
  )                                                                                                                                                                                           
                                                                                                                                                                                              
  -- Приоритеты                                                                                                                                                                               
  priorities (                                                                                                                                                                                
      id UUID PRIMARY KEY,                                                                                                                                                                    
      user_id UUID REFERENCES users,                                                                                                                                                          
      education INT,                                                               
      career INT,                        
      health INT,                                                                                                                                                                             
      leisure INT,                       
      social INT,                                                                                                                                                                             
      household INT,                                                               
      updated_at TIMESTAMP               
  )                                      
                                
  -- Активности                          
  activities (                                                                                                                                                                                
      id UUID PRIMARY KEY,                                                                                                                                                                    
      user_id UUID REFERENCES users,                                                                                                                                                          
      title VARCHAR,                                                                                                                                                                          
      description TEXT,                                                                                                                                                                       
      category VARCHAR,  -- education, career, health, etc.                                                                                                                                   
      start_time TIMESTAMP,                                                                                                                                                                   
      end_time TIMESTAMP,                                                                                                                                                                     
      location VARCHAR,                                                                                                                                                                       
      completed BOOLEAN,                                                                                                                                                                      
      created_at TIMESTAMP                                                         
  )                                                                                                                                                                                           
                                                                                                                                                                                              
  -- События города                                                                                                                                                                           
  events (                                                                                                                                                                                    
      id UUID PRIMARY KEY,                                                                                                                                                                    
      title VARCHAR,                                                                                                                                                                          
      description TEXT,                                                                                                                                                                       
      category VARCHAR,                                                                                                                                                                       
      start_time TIMESTAMP,                                                                                                                                                                   
      end_time TIMESTAMP,                                                                                                                                                                     
      location VARCHAR,                                                                                                                                                                       
      latitude FLOAT,                                                                                                                                                                         
      longitude FLOAT,                                                                                                                                                                        
      source VARCHAR,  -- 'researchinspb', 'leti', etc.                            
      external_id VARCHAR,                                                                                                                                                                    
      cached_at TIMESTAMP                                                                                                                                                                     
  )                                                                                                                                                                                           
                                                                                                                                                                                              
  -- Достижения                                                                                                                                                                               
  achievements (                                                                                                                                                                              
      id UUID PRIMARY KEY,                                                                                                                                                                    
      name VARCHAR,                                                                                                                                                                           
      description TEXT,                                                                                                                                                                       
      icon VARCHAR,                                                                                                                                                                           
      points INT,                                                                                                                                                                             
      criteria JSONB                                                                                                                                                                          
  )                                                                                                                                                                                           
                                                                                                                                                                                              
  -- Достижения пользователей                                                                                                                                                                 
  user_achievements (                                                              
      id UUID PRIMARY KEY,                                                                                                                                                                    
      user_id UUID REFERENCES users,                                                                                                                                                          
      achievement_id UUID REFERENCES achievements,                                                                                                                                            
      unlocked_at TIMESTAMP                                                                                                                                                                   
  )                                                                                                                                                                                           
                                                                                                                                                                                              
  -- Баллы и статистика                                                                                                                                                                       
  user_stats (                                                                                                                                                                                
      id UUID PRIMARY KEY,                                                                                                                                                                    
      user_id UUID REFERENCES users,                                               
      total_points INT,                                                                                                                                                                       
      current_streak INT,                                                                                                                                                                     
      longest_streak INT,                                                                                                                                                                     
      balance_score FLOAT,                                                                                                                                                                    
      updated_at TIMESTAMP                                                                                                                                                                    
  )
