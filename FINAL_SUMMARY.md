   - Генерация персонализированного плана дня

## 🚀 Как запустить

### Быстрый старт
```powershell
.\quick-start.ps1
```

### Или вручную

**Backend:**
```bash
cd backend
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
python init_db.py
uvicorn main:app --reload
```

**Frontend:**
```bash
cd frontend
npm install
npm run dev
```

**Тесты:**
```bash
cd frontend
npm run test
```

## 🔧 Технические детали

### API Endpoints
```
POST   /api/auth/register       - Регистрация
POST   /api/auth/login          - Вход
GET    /api/user/profile        - Профиль
PUT    /api/user/priorities     - Обновить приоритеты
GET    /api/user/priorities     - Получить приоритеты
POST   /api/plan/generate       - Сгенерировать план
GET    /api/plan/daily          - План дня
POST   /api/plan/chat           - Чат с AI
GET    /api/events/             - События
GET    /api/events/{id}         - Детали события
POST   /api/activities/         - Создать активность
GET    /api/activities/         - Список активностей
POST   /api/activities/complete - Отметить выполнение
GET    /api/activities/stats    - Статистика
```
